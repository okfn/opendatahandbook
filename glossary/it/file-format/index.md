title: Formato di file
---

La descrizione di come un {file} Ã¨ rappresentato sul disco di un computer. Il formato generalmente corrisponde all'ultima parte del nome del file ('estensione'), es. un archivio in un file in un formato [CSV](/glossary/it/terms/csv/) potrebbe essere chiamato lista-delle-scuole.csv. Il formato di file fa riferimento al formato interno del file, non come invece mostrato rispetto agli utenti. Es. file CSV e {XLS} hanno una struttura molto diversa, ma appaiono molto simili se non identici se aperti in un [foglio di calcolo](/glossary/it/terms/spreadsheet/) come Excel.
